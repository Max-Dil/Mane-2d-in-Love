In mane, all objects are stored in groups.
Groups allow you to manage a large number of objects,
they can be used to create both scenes and cameras.

When initializing the game mane will already create a group, it will be
in mane.display.game

Set background color
mane.display.setBackGroundColor(1,1,0,1) -- r, g, b, a

Wireframe rendering mode
mane.display.wireframe = true -- default false

Drawing mode for all physical bodies
mane.display.renderMode = “hybrid” -- default “normal”

Shift all graphics:
mane.display.setShear(100, 100) -- kx, ky
